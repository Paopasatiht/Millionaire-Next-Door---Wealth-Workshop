  • get_paw_status(actual, expected)

  • fetch_all(sql, params)

  • build_dashboard_sheet(wb, year)
date_helpers.py
  • current_year() → int
  • current_month() → int
  • months_list() → list[str]
  • years_available(db) → list[int]
Request Flow — ตัวอย่างเมื่อกรอก Monthly Log
User action กรอก Monthly Log กด Save
HTTP POST /monthly → src/routes/monthly.py
Utils monthly.py → utils/db_helpers.upsert() + utils/paw.calc_savings_rate()
DB db_helpers → data/mnd-data.db → redirect → GET /monthly
