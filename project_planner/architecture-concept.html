Pages / Routes

หน้าจอที่จะมีใน localhost:5000

🏠
GET /

Dashboard

PAW/UAW status, savings rate, net worth trend, KPI รายปี

💰
GET/POST /budget

Annual Budget

กรอก/แก้ budget รายหมวด ดู actual vs budget

📅
GET/POST /monthly

Monthly Log

กรอกรายเดือน — รายได้ รายจ่าย net worth note

🎯
GET/POST /goals

Goals Matrix

เพิ่ม/แก้/ติดสถานะเป้าหมายทั้ง 5 horizon

⏱️
GET/POST /planning

Planning Time

บันทึกชั่วโมงวางแผนการเงินรายเดือน

📝
GET/POST /review

Annual Review

ตอบคำถามทบทวนประจำปีทีละข้อ

📤
GET /export

Export Excel

ดึงข้อมูลจาก DB → สร้างไฟล์ .xlsx ให้ download

Startup

วิธีเปิดใช้งาน

# ติดตั้งครั้งแรก
pip install flask openpyxl

# รันทุกครั้งที่จะใช้
python app.py

# แล้วเปิด browser ไปที่
http://localhost:5000
ก่อน Code

3 จุดที่ต้อง confirm ก่อน

1
Port และ year default ใช้ port 5000 ได้เลย? และ default year ควรดึงจาก system date หรือให้ user เลือกเองทุกครั้ง?
2
DB อยู่ที่ไหน สร้าง data/mnd-data.db ที่ root repo ได้เลย หรืออยากเลือก path เองตอน startup?
3
เริ่ม build จากหน้าไหนก่อน แนะนำ: Dashboard + Monthly Log ก่อน เพราะเป็น core loop รายเดือน — เห็นผลเร็ว แล้วค่อยต่อ Budget, Goals ทีหลัง
